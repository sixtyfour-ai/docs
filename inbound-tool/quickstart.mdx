# Quick Start Guide

This guide will help you set up and start using the Inbound Tool quickly.

## Prerequisites

Before you begin, make sure you have:

- A SixtyFour AI account
- API credentials
- Node.js installed (version 14 or higher)

## Installation

1. Install the Inbound Tool package:

```bash
npm install @sixtyfour/inbound-tool
```

2. Import the package in your project:

```javascript
import { InboundTool } from '@sixtyfour/inbound-tool';
```

## Basic Usage

Here's a simple example to get you started:

```javascript
const inboundTool = new InboundTool({
  apiKey: 'your-api-key',
  environment: 'production'
});

// Initialize the tool
await inboundTool.initialize();

// Start processing leads
const results = await inboundTool.processLeads({
  source: 'linkedin',
  filters: {
    industry: 'technology',
    companySize: 'medium'
  }
});
```
